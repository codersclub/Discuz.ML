+======================================+
   Discuz! X3 Beta 20130305
+======================================+
新增 移动浏览全方位适配。
新增 面向非智能机的低端wap版，面向iphone、安卓高端机的高端触摸版
新增 独立移动客户端（iphone/android）支持。客户端API功能更加丰富，一键生成属于自己论坛的独立客户端
新增 门户资讯页面静态化，同时支持URL别名自定义。在增强服务器负载的同时，优化站点的SEO能力。
新增 论坛支持https、IPV6安全链接访问 ，让特殊环境下的论坛数据吞吐更加安全
新增 版块的收藏人数、发帖数排行、以及每日版块发帖数排名变化，增加版主攀比动力，让版块活跃起来
新增 游客查看主题的时候不再显示额外用户数据，大幅降低数据库查询数
新增 论坛首页四格标配，让论坛的热帖、新帖等可以更好的展示给用户
新增 优化大师，全面检测论坛的安全性和性能优化设置，引导站长对论坛进行更好的优化调整
新增 针对指定版块的主题、资讯、日志进行防采集设置，让采集本站内容不再那么容易
新增 对帖子查看页面用户资料信息根据用户组进行单独设置，让不同的用户组，显示的资料不一样，更显独特身份
新增 控制注册新用户的时候，是否邮箱必填
新增 可以对版块回复框进行背景图片设置。这可以更有利于版规需知的传播和阅读
新增 可以针对一些特殊版块，控制用户是否需要使用积分购买才可进入
新增 可以批量修改用户的用户组
新增 可以对帖子查看页面用户头像下面的参数进行自由调整
新增 可以对指定的版块、群组，进行主题直播推广，让热门活动更具及时性和参与性
新增 浏览帖子的时候，右下角增加快速跟帖、快速返回列表、快速置顶的操作菜单
新增 使用免费的道具，不再需要购买后才可使用。诸如如果设置涂鸦板为免费道具，无需折腾，直接就可以使用
新增 发布主题的时候，可以使用目录浮动引导功能。让具有多章节的主题（比如小说站）更加适合阅读
新增 发布主题的时候，可以对帖子内容进行加密，让知道密码的用户才可以查看
新增 用户的名片，按照性别使用不同的颜色显示，男女不同更明显
新增 显示用户上次登录的IP、时间提醒，让帐号更加安全
新增 查看主题的时候，可以查看热门回帖、可以屏蔽指定人的回帖、可以屏蔽水帖，让看帖体验更加舒畅
新增 查看主题的时候，可以选择“精简”模式，并可自由切换。精简模式下，内容为王
新增 发起投票的时候，可以同时上传图片，从而针对图片进行投票
新增 版块列表、看帖页面的用户名的颜色，根据用户组设定，进行不同的展示，用户身份更加突出
新增 查看主题的图片的时候，可以将好的图片收藏到自己的相册里面
新增 版块列表自动加载下一页，同时会剔除可能重复的主题，让翻页操作成为过去式
新增 引导游客注册，主题里面的精彩图片，默认只显示缩略图，需要成为会员才可以查看大图
新增 发布主题的时候，可以将主题内容自动成为图片（用户组权限控制）
新增 增加图酷功能，把帖子中的图片集中到一起查看，读图时代到来
新增 有新提醒的提示改进，提醒分类，更加便于查看提醒
