+--------------------------------------------------------------------------+
 Discuz! X Community Software Installation
+--------------------------------------------------------------------------+

1. Upload all the files from /upload directory to your server

2. Set the directory properties (windows server can ignore this step).
   The following directories require read and write permissions:
	./config
	./data (subdirectory containing the data)

3. Run the installation script /install/
   Please run the install in the browser program,
   by visiting the address:
   http://your_domain/forum_catalog/install/

4. Follow the installation prompts and tips
   until the installation is completed.
